# 📚 Sistema de Gestión Académica

Proyecto **full-stack** desarrollado con tecnologías modernas para la gestión académica, permitiendo la administración de estudiantes, cursos y procesos educativos.

---

## 🚀 Tecnologías utilizadas

- ⚙️ Backend: NestJS  
- 💻 Frontend: Next.js  
- 🗄️ Base de datos: PostgreSQL  
- 🐳 Contenedores: Docker  
- 🔗 ORM: Prisma

## 📌 Descripción del proyecto

Este sistema de gestión académica tiene como objetivo centralizar y optimizar la administración de información educativa. Se implementa una arquitectura full-stack moderna, integrando backend, frontend y base de datos en un entorno escalable y mantenible.
